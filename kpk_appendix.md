# KPK 知识点：附录知识

## KPK-09：成人肥胖患者的食物选择（附录1）

| 字段 | 内容 |
|------|------|
| **KPK ID** | KPK-09 |
| **核心知识** | 科学合理选择食物对于成功减重至关重要。在控制总能量基础上注意多样化与合理化，保证平衡膳食 |
| **适用人群** | 成人肥胖人群 |
| **食养建议** | **谷薯类（每日150～300g）：** 全谷物和杂豆50～150g（如黑米、玉米等营养素密度较高的），薯类50～100g；粗细搭配，避免长期单纯摄入粗粮或细粮。**蔬菜类（300～500g/d甚至更多）：** 深色蔬菜占1/2以上（深绿色：菠菜/油菜等；橘红色：胡萝卜/西红柿；紫红色：紫甘蓝/红苋菜等）；每天至少3～5种。**水果类（约200g/d）：** 减少高糖水果（榴莲/香蕉/荔枝/鲜枣等），不宜饮用果汁。**肉类/水产品/蛋类：** 优先选高蛋白低脂肪——水产品每周≥2次或总量280～525g（40～75g/d）；畜禽肉每周≤500g（约≤70g/d），选纯瘦肉，不建议完全不吃红肉；蛋类每周280～350g（≤7个蛋）。**奶豆类：** 低脂/脱脂牛奶300～500mL/d（乳糖不耐受可选无糖低酸奶或无乳糖产品）；大豆及制品每日15～25g，避免油炸及高盐豆制品。**坚果类：** 每周平均50～70g（约10g/d），首选原味，能量计入三餐总量。此外警惕"看不见"的脂肪（肉类/内脏/坚果中的隐形脂肪）。 |
| **食材/方剂** | 详见附录1表1.1各类食物选择举例（优选/限量/不宜三类） |
| **注意事项** | 各食物类别均有优选、限量、不宜的具体清单（共9大类：谷薯类/蔬菜类/水果类/畜禽类/水产类/豆类/蛋乳类/饮料类/坚果类）。 |
| **关联KPK** | KPK-01（控制总能量），KPK-02（少高能量食物），KPK-05（食药物质选择） |

## KPK-18：常见身体活动强度系数（附录6）

| 字段 | 内容 |
|------|------|
| **KPK ID** | KPK-18 |
| **核心知识** | 以代谢当量(MET)表示活动强度，数值代表活动时能量消耗相当于安静时的倍数。1MET=每分钟每千克体重消耗3.5mL氧气 |
| **适用人群** | 成人肥胖人群及运动处方制定者 |
| **食养建议** | **强度分类：** 静态行为（≤1.5MET）：安静躺/坐/站；低强度（1.6～2.9MET）：读书/打字/洗涤/拖地/步行3km/h等；中等强度（3.0～5.9MET）：购物/骑行4km/h/跑步5km/h/瑜伽/八段锦/太极/乒乓球/广播体操等；高强度（≥6.0MET）：跑步6km/h+/登山/篮球/羽毛球/跳绳/有氧健身操/拉丁舞等。**基础代谢率公式：** 女性=655+9.5×体重(kg)+1.8×身高(cm)-4.7×年龄(岁)；男性=66+13.7×体重(kg)+5.0×身高(cm)-6.8×年龄(岁)。**活动能量消耗计算：** MET值×体重(kg)×活动时长(h) kcal。**总能耗=基础代谢率×非活动时间占比+活动能耗+食物热效应（约10%）。**引用标准：《健康成年人身体活动能量消耗参考值》(T/CSSS 002-2023)。 |
| **食材/方剂** | 无 |
| **注意事项** | MET值适用于中国18～64岁健康成年人。职业性劳动也列入活动强度分类（农业/消防/造船厂/矿山工作等）。 |
| **关联KPK** | KPK-04（多动少静），KPK-06（安全减重） |
