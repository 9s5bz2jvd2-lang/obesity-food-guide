## 指南出处

- **全称**：《成人肥胖食养指南（2024年版）》
- **发布单位**：国家卫生健康委办公厅
- **制定依据**：《健康中国行动（2019—2030年）》《国民营养计划（2017—2030年）》

## 功能特点

- **超重肥胖判定**：BMI计算+腰围测量，精准定位肥胖等级
- **能量分级管理**：3个能量级别（1400/1600/1800kcal）个体化膳食方案
- **食养原则**：6条官方原则的详细解读与实操建议
- **中医食养**：5种证型（胃热火郁/痰湿内盛/气郁血瘀/脾虚不运/脾肾阳虚）+ 食养方
- **地区食谱**：7大地区×4季节×3能量等级完整食谱，含能量标注
- **食物交换表**：7类食物等量交换，灵活配餐
- **科普风格**：说人话、给具象量、指误区——精准但不端着

## 关键数据速查

| 项目 | 推荐量 | 通俗理解 |
|------|--------|---------|
| 能量摄入 | 1400-1800kcal/天（按级别） | 个体化定制，不搞一刀切 |
| BMI分级 | 超重≥24，肥胖≥28 | 算算你的BMI：体重(kg)/身高(m)² |
| 腰围风险 | 男≥90cm，女≥85cm | 肚子胖比全身胖更危险 |
| 烹调油 | ≤25g/天 | 两汤匙封顶 |
| 食盐 | <5g/天 | 一个啤酒瓶盖 |
| 运动 | ≥150min/周中等强度 | 每天快走30分钟就够了 |
| 减重速度 | 0.5-1kg/周 | 别图快，稳才是真 |
