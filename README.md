# 成人糖尿病食养助手 (Diabetes Food Guide Skill)

基于国家卫生健康委《成人糖尿病食养指南（2023年版）》的AI科普对话助手。

## 概览

| 项目 | 内容 |
|------|------|
| 指南全称 | 成人糖尿病食养指南（2023年版） |
| 发布机构 | 国家卫生健康委 |
| 版本 | v1.0.0 |
| 作者 | 王润圆 |

| 食谱数量 | 84套（7地区×4季节×3套） |

| 食养方 | 9个 |

## KPK知识体系

### 食养原则（KPK-01~08）

| KPK | 标题 |
|-----|------|
| KPK-01 | 食物多样，养成和建立合理膳食习惯 |
| KPK-02 | 能量适宜，控制超重肥胖和预防消瘦 |
| KPK-03 | 主食定量，优选全谷物和低血糖生成指数食物 |
| KPK-04 | 积极运动，改善体质和胰岛素敏感性 |
| KPK-05 | 清淡饮食，限制饮酒，预防和延缓并发症 |
| KPK-06 | 食养有道，合理选择应用食药物质 |
| KPK-07 | 规律进餐，合理加餐，促进餐后血糖稳定 |
| KPK-08 | 自我管理，定期营养咨询，提高血糖控制能力 |

### 附录知识（KPK-09~12）

| KPK | 标题 |
|-----|------|
| KPK-09 | 附录1：成人糖尿病患者的食物选择（GI分类表+证型食药物质推荐） |
| KPK-10 | 附录2：常见食物交换表（7类食物等量交换） |
| KPK-11 | 附录4：成人糖尿病食养方举例（9个中医食养方） |
| KPK-12 | 附录5：糖尿病诊断标准（糖代谢状态分类+诊断标准） |

### 疾病背景与问答（KPK-13~14）

| KPK | 标题 |
|-----|------|
| KPK-13 | 疾病背景（疾病定义与流行病学、WHO分类、中医辨证分型） |
| KPK-14 | 问答版实用信息整合（膳食注意事项、中医食养指导、自我管理） |

## 中医证型

| 证型 | 关键症状 |
|------|----------|
| 阴虚热盛 | 烦渴多饮，咽干舌燥，多食善饥，溲赤便秘，舌红少津苔黄，脉滑数或弦数 |
| 气阴两虚 | 倦怠乏力，心慌气短，盗汗自汗，口干舌燥，多饮多尿，五心烦热，大便秘结，腰酸膝软，舌淡或舌红暗、舌边有齿痕，苔薄白少津或少苔，脉细弱 |
| 阴阳两虚 | 乏力自汗，形寒肢冷，腰酸膝软，耳轮焦干，多饮多尿，或浮肿少尿，或五更泻，阳痿早泄，舌淡苔白，脉沉细无力 |

## 使用说明

在 WorkBuddy 中安装此 Skill 后，可直接对话咨询糖尿病膳食管理相关问题。

## 许可

基于国家卫生健康委公开发布的食养指南进行学术蒸馏，仅供健康科普参考。
