# 成人肥胖食养助手 (obesity-food-guide) v1.0.0

> 基于国家卫生健康委《成人肥胖食养指南（2024年版）》蒸馏的AI科普对话Skill

## 简介
本Skill将卫健委官方指南蒸馏为结构化KPK知识体系，包含18个知识点、84套食谱、10个食养方、5种中医证型辨证。

## 核心数据

| 项目 | 数量 |
|------|------|
| KPK知识点 | 18个 |
| 食谱 | 84套（7地区×4季×3能量） |
| 食养方 | 10个（5证型各2-3个）|
| 中医证型 | 5种 |
| 能量等级 | 1200/1400/1600 kcal |

## 文件结构

| 文件 | 说明 |
|------|------|
| SKILL.md | Skill元信息与能力说明 |
| system_prompt.md | 系统提示词 |
| skill.yaml | 结构化配置参数 |
| knowledge_base.md | 知识库总入口索引 |
| kpk_principles.md | KPK-01~06 食养原则 |
| kpk_disease.md | KPK-14~16 疾病背景 |
| kpk_appendix.md | KPK-09~13,18 附录类 |
| kpk_qa.md | KPK-17 常见问题解答 |
| recipes_data.md | 84套完整食谱数据 |
| dietary_formulas.md | 10个中医食养方 |

## 使用方式
安装后即可在对话中触发，支持BMI判定、证型辨证、食谱推荐、食养方查询等场景。

## 数据来源
国家卫生健康委《成人肥胖食养指南（2024年版）》
