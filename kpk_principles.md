# KPK 知识点：食养原则

## KPK-01：控制总能量摄入，保持合理膳食

| 字段 | 内容 |
|------|------|
| **KPK ID** | KPK-01 |
| **核心知识** | 控制总能量摄入和保持合理膳食是体重管理的关键。通过能量负平衡达到减重、减少体脂的目标，同时保障营养素充足摄入 |
| **适用人群** | 成人肥胖人群（BMI≥28.0kg/m²）及超重人群（24.0≤BMI<28.0kg/m²） |
| **食养建议** | **能量控制方法（任选其一）：** ①基于不同人群每天的能量需要量，推荐每日能量摄入平均降低30%～50%或降低500～1000kcal；②男性1200～1500kcal/d、女性1000～1200kcal/d的限能量平衡膳食；③超重个体给予85%摄入标准，肥胖个体给予80%摄入标准；④理想体重(kg)=身高(cm)-105，再乘以能量系数15～35kcal/kg（卧床15/轻活动20-25/中活动30/重活动35）。**宏量营养素供能比：** 脂肪20%～30%，蛋白质15%～20%，碳水化合物50%～60%。**三餐分配：** 早:中:晚=3:4:3。主食以全谷物为主（至少占谷物的一半）；蔬菜300～500g/d（深色占1/2以上），水果约200g/d（减少高糖水果）；动物性食物优先选瘦肉、去皮鸡胸肉、鱼虾；奶类选低脂或脱脂。必要时可在专业人员指导下选用高蛋白膳食、低碳水化合物膳食、间歇式断食或营养代餐等。 |
| **食材/方剂** | 全谷物（黑米、玉米、糙米、燕麦、藜麦、小米、荞麦等）、新鲜蔬菜（深色为主）、低糖水果（柚子、蓝莓、草莓、苹果、樱桃等）、瘦肉、去皮禽肉、鱼虾、低脂/脱脂奶类 |
| **注意事项** | 能量摄入需高于人体基础代谢率的基本需求。孕妇、乳母、老年人及患有慢性代谢性疾病的人群应在医生或营养指导人员指导下进行。《中国居民膳食营养素参考摄入量2023版》为膳食能量需要量的参考依据。 |
| **关联KPK** | KPK-02（少高能量食物），KPK-03（纠正不良饮食行为），KPK-06（安全减重） |

## KPK-02：少吃高能量食物，饮食清淡，限制饮酒

| 字段 | 内容 |
|------|------|
| **KPK ID** | KPK-02 |
| **核心知识** | 减少高能量食物（≥400kcal/100g）摄入有助于控制总能量；严格限制脂肪/油/盐/添加糖和酒精是减重的关键措施 |
| **适用人群** | 成人肥胖及超重人群 |
| **食养建议** | **高能量食物定义：** 提供400kcal/100g以上能量的食物，如油炸食品、含糖烘焙糕点、糖果、肥肉等。应少吃这类食物，多吃富含膳食纤维的全谷物和蔬菜。**限制量标准：** 食盐≤5g/d，烹调油20～25g/d，添加糖最好控制在25g以下。**烹饪方式：** 多选用蒸、煮、熘及水滑等方式，少油煎炸。**采购原则：** 主动阅读食品营养标签，选择脂肪、碳水化合物/糖、钠含量低的食物；尽量不选油炸食品、加工肉制品、含糖烘焙糕点、蜜饯、糖果、冰淇淋及含糖饮料。**限酒：** 每克酒精产生约7kcal能量（远高于碳水4kcal/g和蛋白4kcal/g），且几乎无有用营养素——减重期间严格限制饮酒。 |
| **食材/方剂** | 宜选：全谷物、蔬菜、水果等低能量食物。忌选：油条、炸薯条、方便面、奶油蛋糕、油炸果蔬脆、含糖水果罐头、果脯、加工肉类、含糖饮料 |
| **注意事项** | 警惕"看不见"的脂肪——肉类、动物内脏、坚果中均含有较多容易被忽视的脂肪，易导致膳食脂肪过量。坚果能量计入三餐总能量中。 |
| **关联KPK** | KPK-01（控制总能量），KPK-09（附录1食物选择表） |

## KPK-03：纠正不良饮食行为，科学进餐

| 字段 | 内容 |
|------|------|
| **KPK ID** | KPK-03 |
| **核心知识** | 科学选择、进餐规律、定时定量、养成良好饮食行为是维持健康体重的基础 |
| **适用人群** | 成人肥胖及超重人群 |
| **食养建议** | **时间规律：** 保持一日三餐时间相对固定，避免过度饥饿引起饱食中枢反应迟缓而导致进食过量。重视早餐，不漏餐。晚餐17:00～19:00进食，晚餐后不宜再进食任何食物（可饮水）。**行为规范：** 不暴饮暴食；控制随意进食零食、饮料；避免夜宵。在家或外就餐均应饮食有节制、科学搭配，进行标准化定量的营养配餐。**进餐技巧：** ①细嚼慢咽——摄入同样食物时减少总食量；②减缓进餐速度——增加饱腹感，降低饥饿感；③改变进餐顺序——按蔬菜→肉类→主食顺序进餐，减少高能量食物进食量。**特殊情况：** 如饮水后仍饥饿难忍或有低血糖风险者，可适当选择少许低能量高膳食纤维食物。 |
| **食材/方剂** | 无特殊食材，重点在于饮食行为的矫正 |
| **注意事项** | 合理计划每日餐次和能量分配来安排全天膳食是关键。进餐习惯的改变需要时间和坚持。 |
| **关联KPK** | KPK-01（控制总能量），KPK-02（少高能量食物） |

## KPK-04：多动少静，睡眠充足，作息规律

| 字段 | 内容 |
|------|------|
| **KPK ID** | KPK-04 |
| **核心知识** | 身体活动不足和久坐静态生活方式是肥胖发生的重要原因；经常熬夜和睡眠不足可导致内分泌紊乱和"过劳肥" |
| **适用人群** | 成人肥胖及超重人群 |
| **食养建议** | **运动方案（中低强度有氧运动为主+抗阻运动为辅）：** 有氧运动每周150～300分钟中等强度，每周5～7天，至少隔天1次；抗阻运动每周2～3天，隔天1次，每次10～20分钟；每周通过运动消耗2000kcal或以上。**日常习惯：** 培养运动兴趣，把运动变为习惯；增加日常身体活动，循序渐进；每天静坐和被动视屏时间控制在2～4小时以内；长期静坐/伏案工作者每小时起来活动3～5分钟。**睡眠与作息：** 保证每日7小时左右睡眠，建议夜里11点之前上床睡觉；避免熬夜、作息无规律引起的内分泌紊乱和脂肪代谢异常。 |
| **食材/方剂** | 无（本条聚焦于运动与生活方式） |
| **注意事项** | 运动减重贵在坚持，选择喜欢的运动方式，把天天运动融入日常生活。过劳肥与皮质醇等激素紊乱相关，规律作息不可忽视。 |
| **关联KPK** | KPK-01（控制总能量），KPK-06（安全减重），KPK-18（附录6身体活动系数） |

## KPK-05：食养有道，合理选择食药物质

| 字段 | 内容 |
|------|------|
| **KPK ID** | KPK-05 |
| **核心知识** | 遵循"药食同源"理论，结合中医辨证分型论治，针对不同证型选用对应特性的食药物质，辅助减重 |
| **适用人群** | 成人肥胖人群（各中医证型） |
| **食养建议** | **胃热火郁证（清胃热、消导滞）：** 铁皮石斛、麦芽、鲜芦根、荷叶。**痰湿内盛证（化痰消滞）：** 砂仁、芡实、党参、白扁豆、茯苓、山药、莲子、薏苡仁。**气郁血瘀证（理气化瘀）：** 当归†（仅作香辛料/调味品）、桃仁、佛手、砂仁、山楂。**脾虚不运证（健脾益气）：** 山药、莲子、白扁豆、黄芪、麦芽、砂仁、茯苓、木瓜、大枣。**脾肾阳虚证（温阳补虚）：** 小茴香、山药、肉桂、干姜、黄芪。 |
| **食材/方剂** | 详见附录1表1.2（成人肥胖不同证型食药物质推荐） |
| **注意事项** | 1.在限定使用范围和剂量内作为食药物质；2.食用方法请咨询医生、营养指导人员等专业人员；3.†仅作为香辛料和调味品使用。已知对某种食药物质过敏者、正在服用药物可能与食药物质有禁忌时应在专业指导下食用。 |
| **关联KPK** | KPK-10（附录1表1.2证型食药物质），KPK-13（附录4食养方举例），KPK-15（中医分型） |

## KPK-06：安全减重，达到并保持健康体重

| 字段 | 内容 |
|------|------|
| **KPK ID** | KPK-06 |
| **核心知识** | 科学减重需遵照循序渐进的原则，合理设定减重目标与速度；关注体重变化的同时更应监测体脂率和肌肉量 |
| **适用人群** | 成人肥胖及超重人群 |
| **食养建议** | **减重目标：** 6个月内减少当前体重的5%～10%。**减重速度：** 每月减2～4kg（约每周0.5kg）。**三种常见减重曲线：** ①平稳下降型——每周0.5～1kg；②先平后降型——初期1～2个月无明显变化后开始下降；③快后平台型——初期每周1～2kg后进入数周至数月平台期。**自我监测：** 监测体重变化、食物摄入量和身体活动情况；同时关注体脂率和肌肉量变化，减少肌肉流失，维持基础代谢率。**特别人群：** 孕妇、乳母、老年人及慢性代谢性疾病患者须在医生或营养指导下科学减重。**反弹风险：** 过快减重主要为水分丢失而非脂肪减少，恢复饮食后快速反弹。 |
| **食材/方剂** | 无（本条聚焦于减重策略与管理） |
| **注意事项** | 减重并非越快越好——过快减重对机体器官组织造成损伤甚至危及生命。随着非脂肪组织减少，机体对能量变化的反应减弱，需逐步增加能量消耗或进一步限制能量摄入。 |
| **关联KPK** | KPK-01（控制总能量），KPK-04（多动少静），KPK-14（肥胖判定标准） |
