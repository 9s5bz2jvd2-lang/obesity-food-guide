# 成人肥胖食养助手（obesity-food-guide）

## Skill简介

本Skill是基于**国家卫生健康委《成人肥胖食养指南（2024年版）》**蒸馏而成的AI科普对话助手。指南由国民营养健康指导委员会组织编写，面向基层卫生工作者及无合并症的成人肥胖患者。

知识体系包含 **18个KPK知识点**（Teaching Knowledge Point）、**84套地区季节食谱**（7地区×4季×3能量等级）、**10个中医证型食养方**（5种辨证分型各2-3个）以及完整的食物选择与交换表。覆盖从疾病定义、食养原则、中医辨证到具体食谱的完整链条，可直接用于日常减重膳食指导和健康科普。

## 核心能力

| # | 能力 | 说明 |
|---|------|------|
| 1 | 肥胖判定与分型 | BMI/腰围标准、5种中医证型辨证 |
| 2 | 能量与营养素计算 | 1200-1600kcal/d 三档能量配比，宏量营养素精准推荐 |
| 3 | 6条食养原则解读 | 控制能量、少高能食物、纠正饮食行为、运动睡眠、安全减重、食药物质选择 |
| 4 | 84套食谱查询 | 按地区（华北/东北/华东/华中/华南/西南/西北）+季节+能量等级检索 |
| 5 | 中医辨证食疗 | 5种证型对应食药物质及10个经典食养方 |
| 6 | 食物选择与交换 | 宜忌食物清单、同类食物互换参考 |
| 7 | 安全减重指导 | 减重速度、监测指标、禁忌人群警示 |
| 8 | 常见问题解答 | 代餐、断食、补剂等热点问题科学解答 |

## 适用人群

**主要适用：**
- BMI ≥ 28.0 kg/m² 的成人肥胖患者
- 24.0 ≤ BMI < 28.0 的超重人群（预防性干预）
- 关注体重管理的一般健康人群

**不适用/需谨慎：**
- 孕妇、乳母（需遵医嘱）
- 老年人（≥65岁，能量需求不同）
- 合并糖尿病、高血压、心血管病等慢性病患者（需医生指导）
- 进食障碍患者

## 使用示例

```
"我BMI29，属于什么类型的肥胖？该吃什么？"    → 触发KPK-14/KPK-15 + 辨证问诊 + 食谱推荐
"帮我制定一份1400千卡的减肥食谱"             → 触发KPK-01/KPK-12 + 地区/季节确认
"痰湿体质适合喝什么茶？"                      → 触发KPK-05/KPK-13 痰湿内盛证食养方
"一周减3斤可以吗？安全吗？"                   → 触发KPK-06 安全减重速度
"全麦面包和米饭怎么换算？"                    → 触发KPK-09/KPK-11 食物交换
```
