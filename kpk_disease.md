# KPK 知识点：疾病背景与中医分型

## KPK-14：肥胖定义、分类与流行病学

| 字段 | 内容 |
|------|------|
| **KPK ID** | KPK-14 |
| **核心知识** | 肥胖是一种慢性代谢性疾病，根本原因是能量摄入大于能量消耗，多余能量以脂肪形式贮存体内。我国18岁及以上居民超重率34.3%、肥胖率16.4%且呈上升趋势 |
| **适用人群** | 全人群（科普背景知识） |
| **食养建议** | **定义：** 人体脂肪积聚过多达到危害健康程度的慢性代谢性疾病。**病因分类：** ①单纯性肥胖——无明显内分泌/代谢病病因；②继发性肥胖——继发于神经-内分泌-代谢紊乱基础之上。**流行病学数据（《中国居民营养与慢性病状况报告2020》）：** 18岁及以上居民超重率34.3%、肥胖率16.4%。其中18～44岁肥胖率16.4%，45～59岁18.3%，60岁及以上13.6%（呈上升趋势）。2019年全国11.98%心血管疾病死亡归因于高BMI，死亡人数54.95万。预测到2030年成年人超重肥胖率达65.3%，归因医疗费用约4180亿元（占全国医疗总额21.5%）。**健康危害：** 与2型糖尿病、脑卒中、冠心病、高血压、呼吸系统疾病、骨关节炎、胆结石及多种肿瘤的发生相关。 |
| **食材/方剂** | 无 |
| **注意事项** | 肥胖防控已刻不容缓。"减肥关键三分靠动，七分靠吃"。本指南主要面向基层卫生工作者（含营养指导人员）、无合并症和并发症的成人肥胖患者。合并其他疾病者请在医生指导下参考使用。 |
| **关联KPK** | KPK-15（中医认识与分型），KPK-16（附录5判定标准） |

## KPK-15：中医对肥胖的认识及分型

| 字段 | 内容 |
|------|------|
| **KPK ID** | KPK-15 |
| **核心知识** | 中医将肥胖归属于"脂人""膏人""肥人"范畴，记载最早见于《黄帝内经》。属本虚标实证，病位多在脾胃，与肾气虚关系密切，可涉及五脏 |
| **适用人群** | 成人肥胖人群（中医辨证参考） |
| **食养建议** | **5种辨证分型及临床表现：****1. 胃热火郁证：** 肥胖多食，消谷善饥，大便不爽甚或干结，尿黄，口干口苦喜饮水，舌质红苔黄脉数。**2. 痰湿内盛证：** 形体肥胖，身体沉重，肢体困倦，脘痞胸满，可伴头晕口干不欲饮，大便黏滞，嗜食肥甘醇酒，喜卧懒动，舌质淡胖或大白腻/白滑苔，脉滑。**3. 气郁血瘀证：** 肥胖懒动，喜太息，胸闷胁满，面晦唇暗，肢端色泽不鲜甚或青紫，可伴便干失眠、男子性欲下降/阳痿、女子月经不调/量少/闭经/经血色暗有血块，舌质暗或有瘀斑瘀点薄苔，脉弦或涩。**4. 脾虚不运证：** 肥胖臃肿，神疲乏力，身体困重，脘腹痞闷或四肢轻度浮肿（晨轻暮重，劳则尤甚），饮食如常或偏少（既往多有暴饮暴食史），小便不利，大便溏或便秘，舌质淡胖边有齿印薄白/白腻苔，脉濡细。**5. 脾肾阳虚证：** 形体肥胖，易于疲劳，四肢不温甚或厥冷，喜食热饮，小便清长，舌淡胖薄白苔，脉沉细。 |
| **食材/方剂** | 不同证型对应食药物质见KPK-05和KPK-10 |
| **注意事项** | 肥胖病因多与年龄、体质、饮食、情志、劳逸因素有关。辨证涉及痰、湿、热等病理因素，常兼夹痰湿、血瘀、气郁等标实之证。 |
| **关联KPK** | KPK-05（食养有道·食药物质选择），KPK-10（证型食药物质推荐），KPK-12（食谱示例），KPK-13（食养方举例） |

## KPK-16：附录5 — 成人肥胖判定标准

| 字段 | 内容 |
|------|------|
| **KPK ID** | KPK-16 |
| **核心知识** | 我国成人肥胖及中心型肥胖分别以BMI和腰围(WC)进行判定 |
| **适用人群** | 全人群（肥胖筛查与诊断） |
| **食养建议** | **成人体重分类（BMI，kg/m²）：** 肥胖（≥28.0），超重（24.0≤BMI<28.0），正常（18.5≤BMI<24.0），体重过低（<18.5）。**中心型肥胖分类（腰围cm）：** 中心型肥胖前期——85≤男性WC<90 且 80≤女性WC<85；中心型肥胖——男性WC≥90 且 女性WC≥85。**BMI计算公式：** BMI = 体重(kg) / 身高(m)²。**腰围测量：** 腋中线肋弓下缘和髂嵴连线中点的水平位置处体围的周径长度。**引用标准：** 《成人体重判定》（WS/T 428-2013）。 |
| **食材/方剂** | 无 |
| **注意事项** | BMI和腰围是常用判断超重和肥胖程度的指标。除BMI和腰围外，还可结合体脂率、内脏脂肪面积、肌肉量等综合评估（详见原指南附录5完整版）。 |
| **关联KPK** | KPK-14（定义与流行病学），KPK-01（能量控制目标） |
